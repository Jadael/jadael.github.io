T's Musical Tools

A collection of esoteric VCV Rack modules for manipulating seeded RNG and polyphonic signals in interesting ways.

VCV Library GitHub itch.io

Design Philosophy

TMT is a toolkit focused on control voltage manipulation rather than audio generation. The modules feature no knobs, only jacks—all control happens via CV inputs. This design encourages a pure modular patching approach where everything is dynamic and automatable.

While primarily designed for routing signals and working with control voltages, most modules can operate at audio rate for experimental sound design.

The Modules

Spellbook

Tracker-style plaintext sequencer using RhythML notation. Bring the power of text-based composition to your modular patches.

Page

Expander module for Spellbook that extends column capacity for more complex sequences.

Shuffle

Randomly reorder polyphonic signal channels with seedable, deterministic randomization.

Sort

Rearrange and filter polyphonic cables using spreadsheet-like functions and CV control.

Seed

Generate 16 reproducible random voltages from a seed input. Perfect for creating controlled chaos.

Ouroboros

Step sequentially through polyphonic input channels, turning any poly cable into a sequence.

Append

Combine up to 16 input signals into a selectable polyphonic output with CV-controlled subsetting.

Stats

Calculate statistical measures (mean, median, mode, variance, etc.) of polyphonic signals.

Calendar

Generate time and date-based LFO-like control voltages synced to your system clock.

Sight

Oscilloscope with logarithmic, non-linear time scaling for multi-scale signal visualization.

Spine

Provides frequently-used constant CV offsets, inverse, and reverse operations.

Blankt

Resizable blank panel to organize your rack aesthetically.

Key Features

  • Polyphonic Focus: Deep support for VCV Rack's polyphonic cables with tools to manipulate, analyze, and route them
  • Seedable Randomization: Deterministic random generation that can be controlled and reproduced
  • CV-Only Control: No knobs—every parameter is controlled via CV inputs for maximum modularity
  • Text-Based Sequencing: RhythML notation brings the efficiency of tracker interfaces to modular synthesis
  • Open Source: GPL-3.0 licensed, freely available on GitHub

Status

Current version: 2.2.0 (actively maintained)

Available for Windows, macOS, and Linux through the official VCV Rack plugin library.
